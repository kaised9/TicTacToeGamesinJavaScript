## Tic Tac Toe Game

A simple and fun Tic Tac Toe game using HTML, CSS & JavaScript.

## Features
- 2-player game (Same Device)
- Easy to play
- Clean design

## How to Run
1. Download or clone the repo
2. Open `index.html` in your browser
